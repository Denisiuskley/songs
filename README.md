# Песенник — Offline HTML-читалка

Самодостаточный HTML-файл с текстами песен. Работает офлайн, не требует сервера, не запрашивает разрешений.

## Быстрый старт

1. Положите файлы `.docx` или `.txt` с текстами песен в папку `songs/`
2. Запустите сборку:
   ```bash
   python build.py
   ```
3. Получите файл `songbook.html`

## Структура папок

```
text_pril/
├── build.py              # Скрипт сборки
├── songs/                # Исходные тексты
│   └── Песни Купала.docx
└── songbook.html         # Готовый файл (один, самодостаточный)
```

## Формат исходных файлов

Песни внутри `.docx` или `.txt` разделяются строкой из **3+ звёздочек**:

```
Название песни

Текст песни
Строка за строкой

***

Следующая песня
...
```

## Функционал читалки

- **☰ Оглавление** — плавающая кнопка в правом нижнем углу, выезжающее меню с фильтром
- **🔍 Поиск** — поиск по названиям и текстам песен
- **☆ Закладки** — сохранение любимых песен (звёздочка в заголовке)
- **☀ / ☾ Тема** — светлая и тёмная тема
- **A+ / A-** — размер шрифта
- **Автосохранение позиции** — при повторном открытии возвращает к последней песне

## Перенос на телефон

### Android

1. Скопируйте `songbook.html` на телефон (USB, Telegram, email, облако)
2. Откройте файл в Chrome или другом браузере
3. *(Опционально)* Добавьте на главный экран: меню Chrome → "Добавить на главный экран"

### iPhone / iOS ⚠️ Важно

**На iOS файл нельзя просто открыть из приложения "Файлы".** Apple блокирует JavaScript в локальных HTML-файлах начиная с iOS 13. Если открыть файл через "Файлы", вы увидите статичную страницу с сообщением "JavaScript отключен".

**Рабочие способы открыть файл на iPhone:**

1. **Через Safari** — отправьте файл себе в Telegram / почту / облако, откройте ссылку на файл и выберите "Открыть в Safari" (не в предпросмотре).
2. **Приложение Documents (Readdle)** — установите из App Store, откройте файл через Documents → встроенный браузер.
3. **Приложение HTML Viewer** — установите из App Store, откройте файл через него.
4. **Открыть по ссылке (GitHub Pages)** — файл уже размещён на GitHub Pages. Откройте в Safari:  
   `https://denisiuskley.github.io/songs/songbook.html`  
   Это самый надёжный способ. После первого открытия страница сохраняется в браузере и работает без интернета. Можно добавить на главный экран как приложение.

## Добавление новых песен

1. Положите новый `.docx` или `.txt` в `songs/`
2. Перезапустите `python build.py`
3. Заберите обновлённый `songbook.html`

## Автодеплой на GitHub Pages (рекомендуется для iPhone)

GitHub Pages — **бесплатный** хостинг. Файл открывается по ссылке, JavaScript работает на iPhone без ограничений.

### Первичная настройка (один раз)

1. Создайте репозиторий на [github.com/new](https://github.com/new)
2. В папке проекта выполните:
   ```bash
   git init
   git remote add origin https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПО.git
   ```
3. Включите GitHub Pages в настройках репозитория:
   **Settings → Pages → Source → Deploy from a branch → main / root**

### Деплой

```bash
python build.py --deploy
```

Скрипт автоматически:
- Соберёт `songbook.html`
- Сделает `git commit`
- Отправит на GitHub (`git push`)

Через 1–2 минуты файл будет доступен по ссылке вида:
```
https://ВАШ_ЛОГИН.github.io/ВАШ_РЕПО/songbook.html
```

## Работа без интернета (PWA)

`songbook.html` поддерживает **Service Worker**. Это значит:

- **Первое открытие** требует интернет (страница загружается с GitHub Pages)
- **После этого** страница автоматически сохраняется в кэш браузера
- **Без интернета** — открывайте ту же ссылку, песенник работает полностью
- **Закладки, тема, размер шрифта** сохраняются в браузере даже офлайн

### Добавление на главный экран iPhone / Android

После открытия по ссылке:

**iPhone (Safari):**
1. Нажмите кнопку «Поделиться» (квадрат со стрелкой внизу)
2. Прокрутите вниз → «На экран «Домой»»
3. Нажмите «Добавить»

**Android (Chrome):**
1. Меню (три точки) → «Добавить на главный экран»

Песенник откроется как полноценное приложение — без адресной строки, работает офлайн.
